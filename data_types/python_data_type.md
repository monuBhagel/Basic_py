# Python data types

## Python has the following data types built-in by default, <date = march :2025> in these categories : 

| Category    | Data type    |
|-------------|--------------|
|Text         | `str`        |
|Numberic         | `int` , `float` , `complex`        |
|Sequence         | `list` , `tuple` , `range`        |
|Mapping         | `dict`        |
|Set         | `set` , `frozenset`        |
|Boolean         | `bool`        |
|Binary         | `bytes` , `bytearray` , `memoryview`        |
|None         | `NoneType`        |
